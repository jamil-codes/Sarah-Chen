# 💼 Freelancer Portfolio — Sarah Chen

A clean, modern, single-page freelancer portfolio designed for UI/UX designers, brand strategists, and creative professionals. Built to convert visitors into clients.

---

## 📄 Pages & Sections

| Section | Description |
|---|---|
| **Navigation** | Sticky top nav with smooth scroll links and a CTA button |
| **Hero** | Name, title, tagline, two CTA buttons, and live stats (projects, years, clients) |
| **Ticker** | Animated scrolling skill/tool ticker |
| **Selected Work** | Three project case study cards with metrics and mock UI previews |
| **About** | Bio, values, and location |
| **Skills & Tools** | Six-card grid covering Figma, Webflow, Framer, Branding, UX Research, Design Systems |
| **Testimonials** | Three client quotes with avatar and title |
| **Process** | Four-step how-I-work breakdown |
| **Contact** | Service dropdown form + direct email and LinkedIn |
| **Footer** | Social links and copyright |

---

## 🎨 Design

- Dark background with off-white text and selective warm accent highlights
- Italic emphasis on key phrases throughout headings
- Subtle scroll-triggered fade-in animations on every section
- Case study cards feature live mock UI screenshots (fintech dashboard, SaaS analytics, e-commerce store)
- Mobile-first, fully responsive layout

---

## ✏️ How to Customise

To swap in a client's content, update:

- **Name, title, and tagline** — hero section
- **Stats** — projects shipped, years, clients count
- **Project cards** — title, description, metrics, and mock screenshot
- **Bio text** — about section
- **Skills grid** — labels and descriptions
- **Testimonials** — names, quotes, titles
- **Contact links** — email, LinkedIn, Dribbble, Behance

All content is in plain HTML — no build step needed.

---

## 🚀 Deployment

Upload the folder to any static host. Works on Vercel, Netlify, GitHub Pages, or any standard web host.
